Photoshop CC

Photoshop is a powerful and popular image editing software that helps you to apply various effects easily and get consistent results. Many people use this software to create a better quality image, because it has many features that are hardly available in other photo editing software. Most of the regular edits like teeth whitening, skin tone adjusting, and red eye removal can be completed with a single click.

Notepad++

Open Source: I suppose nothing beats having a notepad replacement that is completely free. In addition, its open nature means that changes can be made in order to facilitate one's development efforts. Search and Replace: I've personally found it to be very useful when editing configuration files, which required a search and replace of multiple items.

Visual Studio Code

Visual Studio Code is a lightweight, cross-platform code editor that you can use in Windows, Linux, or OS X platforms to build modern applications for the Web and cloud alike. A preview version of Visual Studio Code is available for free. Visual Studio Code combines the simplicity and power of an elegant modern code editor with support for an integrated navigation and debugging.
